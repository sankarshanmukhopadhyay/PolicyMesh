# v0.15.0 — Norm Compilation and Governance Authoring

This release turns PolicyMesh into a governance authoring and distribution system, not just a policy exchange fabric.

## Added

- first-class `NormSet` and `CompiledPolicyArtifact` models in `links.norms`
- deterministic norm compiler with contradiction detection and provenance metadata
- CLI commands for `links norms init|validate|compile|diff`
- `links policy apply-compiled` to feed compiled policy into the existing apply path
- new schemas for norm sets and compiled policy artifacts
- documentation for the norm engine and norm-to-policy mapping
- example norm sets and compiled outputs for operator workflows

## Why it matters

Earlier releases assumed that policy artifacts already existed. v0.15.0 closes the missing corridor between community governance and policy transport by letting villages formalize structured norms into executable policy.

## Operational notes

- compilation is deterministic and intentionally narrow in scope
- unsupported or contradictory norms fail fast with structured diagnostics
- compiled artifacts preserve lineage to their source norm set and norm IDs
